**The Reestar Website** is a static restaurant website created using HTML and CSS. The project contains two pages and is designed to present restaurant content in a clean and well-structured layout.

HTML is used to build the structure of the web pages, while CSS is used for styling, layout, colors, fonts, and alignment. The website focuses on simplicity and visual clarity, using modern CSS techniques to organize content effectively. No JavaScript or external libraries are used in this project.

This project was developed as a practice project to improve fundamental front-end development skills and understand how multi-page websites are created using basic web technologies.
